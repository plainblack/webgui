Creating language packs TOC

Language packs are simply JavaScript name/value arrays placed in the "<FN Code>.js" files in the "lang" direcotry. Remember to allways use the "lang_" prefix for these value names so that they don't override other variables in the templates. The example below shows how the cut, copy and paste texts are lang packed. Notice there are two kinds of language packs one is the general one shared by all themes these are located in the "jscripts/tiny_mce/langs" directory the other optional one is theme specific language packs these are contained in "jscripts/tiny_mce/themes/<some theme>/langs".

tinyMCELang['lang_cut_desc'] = 'Cut (Ctrl+X)';
tinyMCELang['lang_copy_desc'] = 'Copy (Ctrl+C)';
tinyMCELang['lang_paste_desc'] = 'Paste (Ctrl+P)';
